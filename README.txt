#Covid 19 dashboard
Covid 19 dashboard provides stats of Novel Corona virus in India

# To start application in local

1. mvn spring-boot:run -Dspring-boot.run.profiles=dev
2. We need a postgres instance and DB named testdb

##API reference
https://github.com/covid19india/api
India - https://api.covid19india.org/data.json
State - https://api.covid19india.org/v4/data.json
Sate Daily - https://api.covid19india.org/states_daily.json
